# AI-Crop-Disease-Detector
An AI-powered web application that detects crop diseases from leaf images in real time and provides instant treatment recommendations. Built using Transfer Learning with MobileNetV2, this project is designed to help farmers  especially in agricultural region  identify plant diseases early and take action before crops are damaged.

How It Works:
The user uploads a photo of a crop leaf through the web interface. The AI model analyzes the image and instantly identifies the disease with a confidence score. It then displays recommended treatment advice to help the farmer take immediate action.

Model Performance:
Training Accuracy: 91.2%
Validation Accuracy: 91.9%
Confidence on real images: up to 98.7%
Dataset: PlantVillage (54,000+ images, 38 disease classes)

Tech Stack:
Python
TensorFlow & Keras
MobileNetV2 (Transfer Learning)
Streamlit (Web App)
OpenCV & PIL

Diseases Detected:
Covers 38 disease classes across multiple crops including Tomato, Potato, Corn, Pepper, Strawberry and more — including both diseased and healthy classifications.

Why This Project:
India loses billions of rupees worth of crops every year due to undetected plant diseases. This tool puts the power of AI directly in the hands of farmers — no expertise needed, just a photo of a leaf.
